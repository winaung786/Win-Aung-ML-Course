# Win Aung — ITAI 4374 Machine Learning Course Portfolio

Welcome to my course portfolio for **ITAI 4374: Neuroscience as a Model for AI** at Houston City College. This repository documents my learning journey across machine learning, computational neuroscience, bio-inspired artificial intelligence, and practical Python modeling. The portfolio is organized to make each lab and assignment easy to find, review, and understand.

## About Me

My name is **Win Aung**, and I am interested in how artificial intelligence can be designed from both engineering and biological perspectives. In this course, I explored how ideas from neuroscience, including cortical columns, spiking neurons, sensory processing, memory, attention, and decision-making, can help explain and inspire modern AI systems. My goal is to keep building practical skills in Python, machine learning, robotics, and AI systems while also improving how I communicate technical work clearly.

## Repository Organization

This portfolio follows the required course structure: a main README, a **Labs** folder for hands-on exercises, and an **Assignments** folder for larger reflections and project work. Each folder contains its own README explaining the purpose of the work, the methods used, what I learned, and the challenges I encountered.

| Section | Folder | Description |

| Labs | `Labs/Lab02-Cortical-Column-Voting/` | Python simulation of a simplified cortical-column voting process. |

## Skills Practiced

Throughout the course, I practiced **Python programming**, **simulation**, **technical documentation**, **AI system analysis**, and **professional repository organization**. I also learned to connect abstract neuroscience concepts to concrete machine-learning workflows.

## How to Use This Repository

Each notebook can be opened in Jupyter Notebook, JupyterLab, Google Colab, or VS Code. Most notebooks use standard Python libraries such as `numpy`, `pandas`, and `matplotlib`. The written assignments are provided as Markdown files so they can be read directly in GitHub.

# L04 Simulating Spiking Neurons

## Purpose

This lab models spiking behavior using a basic leaky integrate-and-fire neuron.

## What I Did

I created a small technical artifact that demonstrates the core idea of the lab and documented the reasoning behind the steps. The goal was not to build a perfect scientific model, but to practice connecting neuroscience concepts to computational thinking.

## What I Learned

I learned how membrane potential, threshold, reset, and input current can produce spike-like activity over time.

## Challenges

The main challenge was connecting a mathematical simulation to the biological idea of neural signaling.

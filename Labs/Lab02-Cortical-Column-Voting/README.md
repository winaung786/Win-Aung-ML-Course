# L02 Cortical Column Voting Simulation

## Purpose

This lab explores a simplified model of how cortical columns can combine local evidence and vote on a final interpretation.

## What I Did

I created a small technical artifact that demonstrates the core idea of the lab and documented the reasoning behind the steps. The goal was not to build a perfect scientific model, but to practice connecting neuroscience concepts to computational thinking.

## What I Learned

I learned that distributed decision-making can be represented with small computational units that each process partial evidence before contributing to a group decision.

## Challenges

The main challenge was simplifying a biological idea into code without pretending that the simulation fully represents the brain.

# L05 Bio-Inspired Perception Pipeline

## Purpose

This lab creates a simple perception pipeline inspired by early sensory processing and feature extraction.

## What I Did

I created a small technical artifact that demonstrates the core idea of the lab and documented the reasoning behind the steps. The goal was not to build a perfect scientific model, but to practice connecting neuroscience concepts to computational thinking.

## What I Learned

I learned how raw data can be transformed step by step into features that are easier for a model or decision system to use.

## Challenges

The main challenge was choosing simple features that were understandable while still being meaningful.

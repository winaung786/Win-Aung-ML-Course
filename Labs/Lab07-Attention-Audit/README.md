# L07 A Practice Audit for the Midterm Project

## Purpose

This lab practices auditing an AI system using ideas from attention, memory, and goal-directed behavior.

## What I Did

I created a small technical artifact that demonstrates the core idea of the lab and documented the reasoning behind the steps. The goal was not to build a perfect scientific model, but to practice connecting neuroscience concepts to computational thinking.

## What I Learned

I learned how to evaluate AI systems beyond accuracy by asking what they attend to, what information they retain, and how they support decisions.

## Challenges

The main challenge was turning qualitative neuroscience concepts into practical audit questions.

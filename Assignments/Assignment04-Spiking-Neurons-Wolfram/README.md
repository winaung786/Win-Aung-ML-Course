# Assignment 04 Exploring Spiking Neurons with Wolfram

## Problem

This assignment focuses on spiking neurons, threshold behavior, and the value of computational models for learning neuroscience concepts.

## Approach

I organized the work around a clear question, connected course concepts to practical AI examples, and used professional documentation to explain the reasoning. The focus was on communication, structure, and learning rather than only technical performance.

## Results and Meaning

The assignment helped me understand that AI systems can be evaluated through multiple lenses: accuracy, interpretability, feedback, robustness, and similarity to biological information processing. These perspectives make the work more useful for real-world analysis.

## Reflection

This assignment improved my ability to explain technical ideas in a way that is understandable to both technical and non-technical readers. It also helped me see how neuroscience-inspired AI requires careful thinking because biological systems are complex and cannot be copied directly into software.
